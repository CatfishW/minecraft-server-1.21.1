# Installed Mods (Fabric 1.21.1)

This repository contains the configuration and custom mods for the Minecraft server.

## Utility & Performance
- **Lithium & VMP**: General performance optimizations.
- **FerriteCore**: Memory usage optimization.
- **ModernFix**: Various bug fixes and optimizations.
- **ServerCore**: Server-side performance features.
- **Spark**: Performance profiling tool.
- **TabTPS**: TPS display in the tab list.
- **Krypton**: Network stack optimization.
- **Packet Fixer**: Fixes for large packet issues.
- **Alternate Current**: Optimized redstone implementation.

## Custom & Core Mods
- **Novus Auth**: Custom authentication system.
- **Novus Items**: Custom items and assets.
- **Novus Pay**: Payment integration system.
- **TACZ (Timeless and Classics Zero)**: High-quality gun mod.
- **Easy NPC**: Comprehensive NPC management system.
- **FTB Quests / Teams / Library**: Questing and team management.
- **LuckPerms**: Permissions management.

## Gameplay & Features
- **Essential Commands**: Basic `/home`, `/spawn`, `/warp` etc.
- **Numismatic Overhaul**: Currency and economy system.
- **EconomyCraft**: Additional economy features.
- **Flan**: Land claiming and protection.
- **Mighty Mail**: In-game mail system.
- **Textile Backup**: Automated server backups.
- **Voice Chat**: Proximity voice chat support.
- **Undead Nights**: Zombie survival mechanics.

## Decoration & Building
- **Macaw's Series**: Bridges, Doors, Furniture, Fences, Windows.
- **Adorn**: Furniture and decoration blocks.
- **Beautify**: Aesthetic additions.
- **Chipped**: Massive variety of block textures.
- **Farmers Delight / My Nethers Delight / Corn Delight**: Expanded cooking and farming.
- **Handcrafted**: Detailed furniture.
- **WorldEdit**: Powerful building tool.

## Technical & Libraries
- **Fabric API / Kotlin**: Core modding libraries.
- **Architectury**: Cross-platform library.
- **Cloth Config / owo-lib / Resourceful Lib**: Configuration libraries.
- **Polymer**: Server-side resource pack and entity support.
- **Roughly Enough Items (REI)**: Item and recipe viewer.
- **Mod Menu**: In-game mod list.
- **WaterFrames / WaterMedia**: Media playback capabilities.
- **Entity Model/Texture Features**: Enhanced entity rendering.
