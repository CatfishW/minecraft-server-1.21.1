# Minecraft Server Fabric 1.21.1

Minecraft Fabric server repository for the 1.21.1 version.

## Features
- Optimized performance with Lithium, VMP, and ServerCore.
- Custom gameplay with TACZ guns and Novus series mods.
- Comprehensive questing system via FTB Quests.
- Economy and land protection.

## Getting Started
1. Clone the repository.
2. Run `start.sh` to launch the server.
3. Ensure you have the matching client-side mods (see `MODS.md`).

Detailed mod list can be found in [MODS.md](./MODS.md).
