# 3D NPC Skin Collection (300+ Skins)

This document lists 3D skin URLs for various NPC types, powered by the [Minotar API](https://minotar.net/).
These URLs can be used in your NPC configuration files (e.g., EasyNPC, Citizens, CustomNPCs).

**Usage Format:**
`https://minotar.net/armor/body/<username>/150.png`
(Replace `150` with your desired pixel size, e.g., 50, 100, 300)
