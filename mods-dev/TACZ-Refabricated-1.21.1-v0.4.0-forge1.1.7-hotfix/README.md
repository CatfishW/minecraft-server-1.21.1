## Timeless and Classics Zero: Refabricated

This mod is an unofficial Fabric port of [TaCZ](https://github.com/MCModderAnchor/TACZ).

**Note:**
- **This mod requires [Forge Config API Port](https://modrinth.com/mod/forge-config-api-port).**
- **This mod is still experimental, perhaps there exist some bugs.**
- **<font color="red">If you want to load custom gunpacks in MC1.21.1, you can upgrade gunpacks to be 1.21.1 NeoForge compatible using the [pack upgrader mod](https://www.curseforge.com/minecraft/mc-mods/tacz-pack-upgrader), and then try load them with TACZ-Refabricated-1.21.1.</font>**
- **As for 1.21, you can install [Too Many Shortcuts](https://modrinth.com/mod/too-many-shortcuts) or similar mods to solve keybinding conflicts.**
